Debug Library — developer

Fake devices of every kind webCoRE can target — so a piston someone sends you can be tested even when it uses a device you don't own.

  1. Import the broken piston (front door → Import).
  2. Add the device kinds it uses — click Add below on each one (or Spin up the whole suite for all of them). Each Add creates a real fake device in Home Assistant.
  3. Open the piston in the editor and point its devices at your new test devices (pick them from the device list, like any device).
  4. Drive them on Test Devices — flip the motion on, fire the double-tap, open the garage — and watch the piston react. That's where you see what's broken.

For copies of your own gear (to test your own automations), use the clone list on Test Devices instead.
