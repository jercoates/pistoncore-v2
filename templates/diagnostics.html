Compile target

PistonCore normally compiles each piston to a plain Home Assistant automation and only uses PyScript when the piston needs it. PyScript follows webCoRE's execution model more closely — if a piston compiles but behaves differently than it did on Hubitat, forcing it to PyScript is the escape hatch. Set the default here; override any individual piston below.

Remembered overrides

Overrides are remembered by piston name, so re-importing a piston keeps its setting even though it gets a new internal id. A structural fingerprint is kept too — if it changed, the row says so, so you can tell a genuine re-import from a different piston that happens to share a name.
