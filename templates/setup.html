Set up PistonCore

Four steps. Every one can be skipped and changed later in Settings — nothing here is a one-time decision.

  1. 1 Connect to Home Assistant

    Home Assistant → your profile → Security → Long-Lived Access Tokens.

  2. 2 Where PistonCore writes automations

    PistonCore compiles your pistons into files inside Home Assistant's config folder. It needs a way to write there.

    Home Assistant extras

    Optional. Pistons that speak use this engine.

  3. 3 Let Home Assistant load PistonCore's files

    Home Assistant only loads compiled automations if configuration.yaml points at PistonCore's folders. This shows you the exact lines before anything is written, and takes a timestamped backup first. Requires step 2 to pass.
